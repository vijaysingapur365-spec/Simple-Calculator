# Simple-Calculator
Simple Calculator (CLI) in C is a command-line application that performs basic arithmetic operations like addition, subtraction, multiplication, and division. It features a menu-driven interface, continuous execution, and error handling for division by zero, helping beginners learn core C concepts.

# Simple Calculator in C

This is a command-line calculator written in C.

## Features
- Addition
- Subtraction
- Multiplication
- Division

## How to Run
gcc calculator.c -o calculator
./calculator
